# Escape From Corridor

A first-person horror exploration game inspired by *Shadow Corridor*.  
The player explores a dark corridor, avoids enemies, and collects items to escape.

---

## Overview

Escape From Corridor is a first-person horror game focused on exploration and tension.  
You must carefully move through the map, manage light, and avoid enemies while searching for **Magatama** hidden in the environment.

This project was created for learning purposes using **C++ and DxLib**.

---

## Game Rules

### Game Clear
Collect **3 Magatama** and reach the goal.

### Game Over
If you are caught by an enemy, the game is over.

---

## Controls

| Action | Key |
|------|-----|
| Move | WASD |
| Camera | Mouse |
| Dash | Left Shift |
| Crouch | Left Ctrl |
| Use Ofuda | Left Click |
| Switch Light | R |
| Pick up Magatama | F |
| Quit Game | Esc |
