# EscapeFromCorridor

## Overview
A psychological horror game inspired by ＊Shadow Corridor＊.
The game focuses on fear created through atmosphere, sound, and limited visibility rather than direct combat.

## Concept
- Exploration-based horror experience

## Gameplay
- First-person exploration

## Technology
- Language : C++
- Library  : DxLib

## Development Status
Currently in development.

## Notes
This project is created as a learning and portfolio piece for game programming.
